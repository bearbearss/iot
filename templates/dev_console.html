📖 하드웨어 연동 체크리스트
1
Raspberry Pi — Flask 서버 실행
cd smart_bathroom_dashboard
source venv/bin/activate
python app.py
✔ 이 페이지가 보이면 완료
2
Raspberry Pi IP 확인
hostname -I
예: 192.168.0.15 → ESP32 config.h 의 SERVER_IP 에 입력
3
ESP32 config.h 수정 후 업로드
#define WIFI_SSID "공유기이름"
#define WIFI_PASSWORD "비밀번호"
#define SERVER_IP "192.168.0.15"
Arduino IDE → 업로드 → Serial Monitor 확인
4
ESP32 연결 확인
Serial Monitor 에 [WiFi Connected] 표시
이 페이지 상단에 ESP32 녹색 점 표시
5
센서 데이터 수신 확인
왼쪽 "마지막 ESP32 수신 데이터" 에 실제 값 표시
6
DFPlayer / mmWave / 문센서 테스트
위 빠른 테스트 버튼 → 낙상 시나리오 → 소리 확인
7
외부 접근 (Cloudflare Tunnel)
/tmp/cloudflared tunnel --url http://localhost:5000
출력된 URL → 보호자 스마트폰에서 접속
